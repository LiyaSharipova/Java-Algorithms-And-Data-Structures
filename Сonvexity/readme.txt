Многоугольник задан координатами своих вершин в порядке его обхода. Необходимо проверить является ли многоугольник выпуклым.

input 
5
Введите координаты вершин в порядке обхода 
1 1
1 3
2 2
3 2
3 1
output
Не выпуклый

input
4 
Введите координаты вершин в порядке обхода 
1 1
1 -1
-1 -1
-1 1
output
Выпуклый
