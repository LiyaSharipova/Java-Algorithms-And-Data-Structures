
Перемножение матриц
Реализовать умножение квадратных матриц размерности N x N, где N заключено от 200 до 2000. Умножение должно произодиться в несколько параллельно выполняющихся потоков выполнения количеством m. m <= N. Каждый поток выполнения умножает N/m строк матрицы. Проверить скорость выполнения программы при различных значениях m:
при m=1
при m=2,3,5,10
при m=100
Необходимо выяснить оптимальное количество потоков выполнения для данной размерности матрицы (это будет зависеть от машины)
